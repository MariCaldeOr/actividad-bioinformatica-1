# actividad-bioinformatica-1
1. iniciar sesión en Github
2. asegurarse que mi cuenta de git hub esta configurada en la terminal
3. crear nuevo repositorio
4. generar un token si no se tiene
5. clonar el respositorio en el computador (pegar el link y cd del archivo para descargarlo)
6. confirmar en github
7. actualizar mi archivo (nano y nombre del archivo)
8. git status
9. git add para incluirlo en el commit
10. git commit-m papa preparar los archivos y actualizar el repo
11. git push para subirlo al repo online
